# Real Estate Province Prediction: Comparative Analysis of Machine Learning Models on Property24 Listings

## 📋 Overview

This repository contains the complete code and datasets for the research paper **"Comparative Analysis of Real Estate Prediction Models on Property24 Listings"**. The project implements and compares four machine learning models—Deep Neural Networks, Random Forests, Support Vector Machines, and XGBoost—for predicting the South African province of origin for property listings scraped from Property24.

The key contribution of this work is the investigation of price-based feature limitations for geographic classification and the introduction of a novel positional encoding methodology using Gaussian-distributed coordinate sampling.

run.py, scraper.py, data_preparation.py and property.py form the web scrpaer and analysis.ipynb includes data preprocessing and the models with relevant figures.
